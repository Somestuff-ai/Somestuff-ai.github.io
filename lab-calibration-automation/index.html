🧪 Temperature Calibration Automation

This Python-based application automates temperature sensor data collection in a calibration lab environment. It was successfully submitted for UKAS ISO 17025 accreditation and is used to reduce human error, improve efficiency, and standardize test procedures.

🔧 Libraries Used

  • pyserial — for serial port communication with temperature chambers
  • pandas — for structured data logging and export
  • datetime — to timestamp sensor readings
  • tkinter — for building a user-friendly interface
  • os and time — for automation control and timing

🏆 Achievements & Impact

  • Used in the UKAS audit demonstration for ISO 17025 compliance
  • Reduced manual data entry by 90%
  • Enabled unattended test cycles with consistent and traceable data
  • Trained calibration engineers in software usage
